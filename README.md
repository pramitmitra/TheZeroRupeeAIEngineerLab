# The Zero Rupee AI Engineer

**Building Real Things Before You Graduate**

A hands-on workshop that teaches you how to build real AI applications using only free tools — no GPU, no credit card, no local setup required.

*Instructor: Pramit Mitra *

---

## Workshop Modules

### Module 1 — Demystifying AI
Foundation concepts that cut through the hype. Understand what AI actually is, how large language models work, and where they fit in real-world engineering — without the jargon.

### Module 2 — Prompt Engineering as a Professional Discipline
Prompt design is not about being polite to a chatbot — it's measurable engineering. This module demonstrates how a well-structured prompt produces more reliable output with fewer tokens, and why that matters at scale.

### Module 3 — Building Real Applications Without a GPU
Use a production LLM API (Google Gemini, free tier) to build a working application that accepts unstructured text and returns structured JSON your code can consume — all from a single notebook.

---

## Lab Notebook

The included Jupyter notebook (`The_Zero_Rupee_AI_Engineer_Lab_1.ipynb`) covers both live demos:

- **Part A:** Naive vs. engineered prompt comparison with live token metering
- **Part B:** Resume parsing app that returns schema-enforced JSON using the Gemini API

### Getting Started

1. Open the notebook in [Google Colab](https://colab.research.google.com) (recommended) or locally with Python 3.9+
2. Get a free API key from [Google AI Studio](https://aistudio.google.com/apikey)
3. OpenAI Tokenizer - from https://platform.openai.com/tokenizer
4. Run cells top to bottom

### Free Tools Used

- Google Colab
- Google AI Studio
- Gemini API (free tier)
- `google-genai` Python SDK
